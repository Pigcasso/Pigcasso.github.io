# 本地相片详情页底部显示相片信息

这是果仁相册v0.13.0新增的功能。开启此功能会在进入本地相片详情页后在屏幕底部显示相片的基本信息，目前仅仅是显示相片的宽、高和修改日期。这个功能默认是开启的，可以到设置中关闭。

<img src="../imgs/Screenshot_2019-05-06-23-51-15-518_果仁相册.png" width="270"/>

## 如何关闭底部显示的相片信息

第一步，在主页点击设置

第二步，关闭在屏幕底部显示开关
